# LzStringTool

因為需要跟一些手機專案配合，對方使用了該 library https://pub.dev/packages/lzstring

然後因為一些需求，我需要解密該文字，所以就索性寫了這小工具，不然目前沒有找到可以用的線上解碼的東西，有點惱人..

既然寫了就紀錄分享一下 :)

--

Online compress or decompress for lz-string , base on C# .net core 3.1
